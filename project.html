Design Principles
  • Zero Trust: No secrets in frontend, auth at every layer
  • Defense in Depth: Multiple security layers
  • Fail Securely: Block suspicious activity, log events
  • Cost Optimized: Serverless = pay per request
Key Technologies
  • Frontend: Alpine.js SPA on AWS S3 + CloudFront
  • Security: Cloudflare Workers proxy, Firebase JWT
  • Backend: AWS Lambda, Secrets Manager
  • AI: Google Gemini Pro 1.5 (<$0.01 per analysis)
